
### 题目

[1151. 最少交换次数来组合所有的 1](https://leetcode.cn/problems/minimum-swaps-to-group-all-1s-together/)


### 题解

先统计整个数组中1的数量ones_count, 并将ones_count当作定长滑动窗口的长度. 

对于每一个窗口:如果有x个0,那么代表着要将所有的1连在一起需要将窗口外的x个1换进来,需要进行x次操作.

遍历数组中的每一个窗口, 找到最少的操作次数.


### 代码

```cpp
class Solution {
public:
    int minSwaps(vector<int>& data) {
        int ones_count = 0;
        for (auto& it : data) {
            ones_count += (it == 1);
        }

        int min_op = ones_count;
        int zeros_count = 0;
        for (int l = 0, r = 0; r < data.size(); r++) {
            zeros_count += (data[r] == 0);
            if (r - l + 1 == ones_count) {
                min_op = min(min_op, zeros_count);
                zeros_count -= (data[l++] == 0);
            }
        }

        return min_op;
    }
};
```
