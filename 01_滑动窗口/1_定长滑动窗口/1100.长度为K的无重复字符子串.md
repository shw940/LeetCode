
### 题目

[1100. 长度为 K 的无重复字符子串](https://leetcode.cn/problems/find-k-length-substrings-with-no-repeated-characters/)


### 题解



### 代码

```cpp
class Solution {
public:
    int numKLenSubstrNoRepeats(string s, int k) {
        int num = 0;
        unordered_map<char, int> mp;

        for (int l = 0, r = 0; r < s.size(); r++) {
            mp[s[r]]++;
            if (r - l + 1 == k) {
                num += (mp.size() == k);
                if (--mp[s[l]] == 0) {
                    mp.erase(s[l]);
                }
                l++;
            }
        }
        return num;
    }
};
```
