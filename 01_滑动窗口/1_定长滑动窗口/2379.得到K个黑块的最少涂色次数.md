
### 题目

[2379. 得到 K 个黑块的最少涂色次数](https://leetcode.cn/problems/minimum-recolors-to-get-k-consecutive-black-blocks/)


### 题解



### 代码

```cpp
class Solution {
public:
    int minimumRecolors(string blocks, int k) {
        int min_op = blocks.size();
        int white_bloacks = 0;

        for (int l = 0, r = 0; r < blocks.size(); r++) {
            white_bloacks += (blocks[r] == 'W');
            if (r - l + 1 == k) {
                min_op = min(min_op, white_bloacks);
                white_bloacks -= (blocks[l++] == 'W');
            }
        }

        return min_op;
    }
};
```
