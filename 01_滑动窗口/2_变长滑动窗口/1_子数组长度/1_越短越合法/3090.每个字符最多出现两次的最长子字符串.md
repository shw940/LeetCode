
### 题目

[3090. 每个字符最多出现两次的最长子字符串](https://leetcode.cn/problems/maximum-length-substring-with-two-occurrences/)


### 题解



### 代码

```cpp
class Solution {
public:
    int maximumLengthSubstring(string s) {
        vector<int> mp(200);
        int max_len = 0;

        for (int l = 0, r = 0; r < s.size(); r++) {
            mp[s[r]]++;
            while (mp[s[r]] == 3) {
                mp[s[l++]]--;
            }
            max_len = max(max_len, r - l + 1);
        }

        return max_len;
    }
};
```
